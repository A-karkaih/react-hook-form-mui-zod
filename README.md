# React Hook Form + MUI + Zod Learning Repo

This repository is a learning boilerplate for mastering forms in React using modern tools like React Hook Form, Material UI, and Zod.

---

## 🎯 Goal

The goal of this repo is to learn how to build real-world forms in React like in production applications.

- Manage form state efficiently
- Connect Material UI components with React Hook Form
- Handle validation using Zod
- Build complex and scalable forms

---

## 🧩 What’s inside

This repo contains examples of almost all common form cases, including:

- TextField
- Checkbox
- Radio Group
- Switch
- Autocomplete
- Date Pickers
- Toggle Button Group
- Complex dynamic forms
- Discriminated unions (advanced form logic)

---


## 🚀 Why this repo

Forms in React look simple at first, but in real projects they become complex quickly.

This repo helps us understand:

- How professionals structure forms
- How to keep forms scalable and maintainable
- How to avoid common mistakes in form handling

---

## 📌 Note

The main focus is understanding patterns used in real-world frontend development.
